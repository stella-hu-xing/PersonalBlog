#### Basic Operations
 * Generate new Post
 ```
   hexo new POSTNAME
 ```
 * Delete Post

   *  Delete the post from _source/_post_ folder

 * Run on local host
 ```
   hexo server
 ```
 * Deploy to remote server
 ```
   hexo deploy
 ```
